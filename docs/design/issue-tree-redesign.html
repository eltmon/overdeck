Style-guide v1.2 conformance (v3)

  • Signal law v1.2 throughout: running agents and live convoys are blue (machine working) — matching the StatusDot migration now shipped on main. Emerald appears only as verified outcome (Merged). Amber appears only where a human must act: paused (+ Unpause) and the ready-for-merge gate (+ Merge, the row's one primary CTA).
  • Purple only on live specialist activity (the reviewing convoy icon); the In Review phase badge is neutral unless it is the row's signal.
  • Cyan only on money (per-agent cost, SF Mono tabular).
  • Fonts: DM Sans body; SF Mono for identifiers only (IDs, branches, PR #s, model IDs, costs). font-medium 500 everywhere.
  • Badges: rounded-sm, 8% bg / 32% border, {signal}-foreground text.
  • Restraint: one colored signal per row — edge bar + at most one signal badge; phase/train/labels/resources neutral unless exceptional (CI ✗); pipeline strip neutral fills with only the signal segment colored.
  • Warm neutrals (neutral-400/500), borders white/6%; titles wrap two lines, never clipped.
