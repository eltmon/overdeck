Why this is the bigger bang: it solves the tab-explosion problem and your conflation problem at once. Tabs stay scoped to a Thread (never a 20-tab pile), but Threads are first-class and cheap — switch them like tabs, tag them with #type, and they live one click from Home. The sidebar drops from 18 destinations to Home + what you're actually working on; everything else is a card on Home.

Term Means (no overlap)

#typeLightweight Thread category: #feature · #bug · #explore · #chore.
Tab / PaneA view inside a Thread: Home · agent · terminal · files · commits · plan · docs · web.
