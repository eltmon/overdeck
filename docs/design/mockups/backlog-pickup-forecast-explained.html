Backlog Pickup Forecast — explained

Today the graph arrows mean "depends on". This view re-points them at the question you actually care about: what order will the Flywheel pick this work up, and when? It runs the real pickup rule — an issue is eligible only if it isn't blocked, isn't already running, and has a vBRIEF spec or a PRD draft — then schedules the eligible issues in rank order, a few at a time.

Below uses 10 real issues from your live backlog (not the whole 544) and models 3 concurrent agents, just to make the idea legible. Same engine, fewer cards.
