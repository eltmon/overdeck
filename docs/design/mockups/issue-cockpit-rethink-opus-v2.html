What got moved (and where every old datum lives now)

  • Tree → Agents. One clean row per role: dot · role · model · timer. Click a row → its live feed fills the center (and "Open terminal ↗" launches the pane). The All / Alive / Failed filter is dropped — pointless inside a single issue.
  • The chip plumbing is demoted. workspace · branch local/remote · tmux · vBRIEF · beads · stack · containers · PR · branches all fold into the collapsible Stack drawer at the bottom of the Agents column. Present when you need it, silent when you don't.
  • NOW / ISSUE cards deleted. Phase, Review, PR, CI, Merge-ready were duplicates of the header gates — gone. What survives that the header doesn't show (Diff size, Verification, Next action, Opened/priority) is now one compact facts strip.
  • The center now leads with live work. The Live feed (promoted from the buried Activity tab) answers "what is the agent doing right now" in motion — edits, runs, commits, the review verdict — newest first.
  • Tab strip → "Jump to" deep-links. Review / Test / PR&CI / Plan / Timeline (Activity+History) / Discussion / Costs as quiet chips; Diff / Files / Terminal open as panes (↗). The page no longer opens with a 15-wide menu.
  • Beads rail unchanged — done / working-now / upcoming + a progress bar, the prominent vertical progress column.

Color discipline

red = action required · blue = a machine is working (running agent, active bead, live feed) · emerald = outcome (done) · cyan = money · neutral = everything else. One signal per row · DM Sans 500 · mono only for IDs / branch / model / paths.
