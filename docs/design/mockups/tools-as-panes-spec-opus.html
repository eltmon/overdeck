Decisions before I build

  • 1. Files & Terminal: drop from the tab row, add the right-aligned tool-buttons that open deck panes (mechanism already wired via onOpenPane). This part is trivial + lossless. Good?
  • 2. Conversation — the one judgment call. The “Conversation” tab today holds the issue-level launcher + agent dock + timeline (how you start/talk to agents at the issue level), which is not the same as clicking one agent. Options:
      (a) Make it a tool-button too → opens the issue conversation as a pane (my recommendation — consistent with “tools are panes”).
      (b) Fold the composer into the lane/Overview as a slim “Launch / message” bar, drop the tab.
      (c) Keep Conversation as the one remaining tab (only Files/Terminal leave).
  • 3. Tool set: Files · Terminal · Web (browser). Add Commits/Docs too, or keep to these three?
