Command Deck · Issue view = issue-scoped HomePane

Vocabulary follows pan-1549-workspace-panes.html. Glance-question: "what's the state, and what can I do next?" The old OverviewTab dumped billboard + DAG + 7 tiles + stepper + reviewer grid + tests + PR + sparkline + activity all at once (now hidden in one default-open collapsible). This re-refines it. Draft for alignment — no code.

Principle: the issue HomePane is a cockpit, not a dump. GLANCE = state + next-action. SCAN = the supporting status (reviewers, tests, PR) + the issue's live agents + recent threads. Everything heavy (Plan DAG, PRD/STATE, Beads graph, full cost, discussions) is a pane you open (PlanPane / DocsPane / CommitsPane) or a collapsed section — never crowding the top.
