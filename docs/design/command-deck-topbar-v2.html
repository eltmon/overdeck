What changes vs today
1. The tabs get the entire window width (their own row) instead of sharing a strip with the model picker, deck label, and history icon.
2. The verbose “Deacon is frozen / 1 stopped / Restart All / normal XX% mem” banner collapses into compact status pills on the app bar — click a pill to expand its actions (Resume Deacon, Restart all, etc.).
3. Project switcher moves to the app bar (left), so it’s always visible and the deck doesn’t need its own “Command Deck” label row.
4. Project Activity + Activity Feed become collapsible right-rail toggles (icons), reclaiming horizontal space for the pane + tabs.
5. The model picker tucks to the right of the tab strip (it only affects new spawns), or into the “+” menu.
Tabs remain project-scoped — switching the project switcher swaps the whole tab set.
