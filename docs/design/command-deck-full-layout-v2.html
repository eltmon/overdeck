• Navigate = a slim icon nav rail (Command Deck / Flywheel / Board / Pipeline / Agents / Metrics / Settings…) + the deck tree (Conversations + Issues). The old wide app-nav sidebar becomes the 44px icon rail; hover/expand for labels.
• Work = the full-width tab strip + the active pane (your cockpit, now with real width).
• Awareness = the two feeds you have today (Project Activity + Activity Feed) merge into ONE right rail with Needs-you / Project / Global tabs — instead of two columns each eating ~280px.
